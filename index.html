Генератор счёт-фактур для заказов Korzinka

Загрузите файл вида “Заказы с товарами - АСОСИЙ.xls”. Программа объединяет магазины, суммирует повторяющиеся товары и формирует счёт-фактуру с НДС 12%.

Загрузите Excel-файл

Обычно в файле есть колонки Магазин, Название, Общая сумма, Товар, Кол-во, Цена за единицу, Стоимость, НДС, Стоимость с учётом НДС. Эти заголовки находятся автоматически, после чего строки читаются программой.

    Как работает анализ?

    • Название магазина берётся из колонки Название; одинаковые названия объединяются в один магазин.
    • Товары читаются из строк после заголовка Товар.
    • Значения Кол-во, Цена за единицу, Стоимость, НДС и Стоимость с учётом НДС находятся автоматически; пустые суммы рассчитываются по формуле НДС 12%.
